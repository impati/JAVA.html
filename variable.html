VARIABLE

변수란, "하나의 값을 저장할 수 있는 메모리 영역"

  • 변수의 선언과 초기화
    • 
    public class Main{
        public static void main(String args[]){
                int var1; //선언
          	int var2 = 0; //선언과 동시에 초기화
        	}
        }

  • 변수의 타입

    • 기본형
      • 논리형(boolean) 1byte

        • ->true,false를 값으로 하는 자료형. 조건식,논리식에 사용한다.

      • 문자형(char) 1byte

        • ->문자하나를 저장하는 자료형.

      • 정수형(int,long) (4byte,8byte)

        • ->정수를 다루는 자료형. int는 약 -21억~ 21억 범위의 값을 가질 수 있다.

        ->long은 (-2^63) ~ (+2^63-1) 의 값을 가진다.

      • 실수형(float,double)(4byte,8byte)

        • 실수형 자료형 float의 정밀도는 7자리이며 double의 정밀도는 15자리이다.
