    • BufferedReader를 이용한 입력

      • 1.java.io.BufferedReader , java.io.InputStreamReader 와 throws IOException 선언

      2.클래스 안에서 BufferedReader 객체를 형식에 맞게 생성한다.

      3.자료형에 맞는 멤버함수를 호출하여 반환받아 변수 저장한다.

      4.메인 함수에서 close()를 반드시 호출해준다.

      
    import java.io.BufferedReader;
    import java.io.IOException;
    import java.io.InputStreamReader;
     //훨씬 빠른 입력.
     public class Main {

        public static BufferedReader cin = new BufferedReader(new InputStreamReader(System.in));
        public static void main(String[] args) throws IOException {

              		//cin.readLine()은 개행기준으로 입력받는다.
              		int n = Integer.parseInt(cin.readLine());
              		double d = Double.parseDouble(cin.readLine());
              		String str = cin.readLine();

              		String []s = str.split(" "); //개행단위로 입력받고 구분문자 기준으로 분리하여 다룰 수 있다.

              		cin.close();

              	}
      }

    • BufferedWriter를 이용한 출력

      • 1.java.io.BufferedWriter , java.io.OutputStreamWriter 와 throws IOException 선언

      2.클래스 안에서 BufferedWriter 객체를 형식에 맞게 생성한다.

      3.자료형에 맞는 멤버함수를 호출하여 반환받아 변수 저장한다.

      4.메인 함수에서 close()를 반드시 호출해준다.

      
  import java.io.BufferedWriter;
  import java.io.IOException;
  import java.io.OutputStreamWriter;
  //빠른 출력.
  public class Main {
  	public static BufferedWriter cout = new BufferedWriter(new OutputStreamWriter(System.out));
  	public static void main(String[] args) throws IOException {

  		int var = 10;
  		String s= "impati";
  		double pi = 3.14;
  		//write함수에 대해서 아직 잘 모른다. 정수나 실수일 때 +""를 추가해주어야 출력이된다.
  		cout.write(var+"");
  		cout.write(s);
  		cout.write(pi+"\n");//개행 추가 출력.

  		cout.close();
  	}
  }
